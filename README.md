# mz-project-cli
项目骨架搭建脚手架

## 配置
在config.json中配置项目骨架,ex:
```json
"admin": {
    "gitRepo": "git@github.com:maizi-base/egg-boilerplate-admin.git",
    "blpName": "egg-boilerplate-admin"
  }
```

## 使用
```bash
npm i mz-project-cli -g
mz-project-cli create [project-name]
```

## 帮助
```bash
mz-project-cli -h
```
